Multilineage tracking: 1. Create a directory with the parent label files to be analyzed and pl3004.py2. From within that directory run “python pl3004.py file(1).csv file(2).csv … file(i).csv”. file(1).csv to file(i-1).csv are the files used to generate the mutilineage tracking, which should be in the directory. Provide a name for file(i).csv which is the result the computer will generate. 3. Load the newly generated parent label file on the final mesh and verify that the cells from the same lineage have the same label as the first time point mesh. For example:We named the tracking file between 0hr and 12 hr as 0_12_parentlabel.csv. We have the parent label files 0_12_parentlabel.csv, 12_24_parentlabel.csv, 36_48_parentlabel.csv. Then we run: python pl3004.py 0_12_parentlabel.csv, 12_24_parentlabel.csv, 36_48_parentlabel.csv 0_48_parentlabel.csv. The python script will generate the corresponding parent labels tracking cells between 0 hr and 48 hr: 0_48_parentlabel.csv.
